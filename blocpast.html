Aprendiendo a manejar versiones

Aprenderemos cosas increibles con GitHub, y estos pasos te ayudar a crear un blogpost y manejar las versiones

  • git init "Arranca el repositorio en tu pc"
  • git add nombre-de-tu-archivo
  • git commit -m "mensaje" para guardar los cambios
  • git commit -am "mensaje" //esto lo podemos usar cuando ya se hizo un add . por primer vez
  • git status // nos muestra si hay cambios por guardar
  • git log // nos muestra la historia de cambios que ha tenido el archivo

Se puso otro parrafo en master y este se unira con merge
